## Spec
* GET / -> get all to-do list
* GET /:entryId -> get entryId's to-do
* POST / -> register to-do
* DELETE / -> delete all to-do list
* DELETE /:entryId -> delete entryId's to-do
* PATCH /:entryId -> update entryId's to-do
